# Dependencies Explained

## base

### What is it?

A lightweight, highly optimized industrial-strength alternative to OCaml's standard library.

### Link please?
https://github.com/janestreet/base

## core

An extension of the standard library extension built on top of `base`. It hads more utilities.

### Link please?
https://github.com/janestreet/core

## async

A concurrent execution library. I probably will use this at the api boundary layer.

### Link please?
https://github.com/janestreet/async

## ppx_jane

A collection of metaprogramming syntax rewriters.

### Link please?
https://github.com/janestreet/ppx_jane

## caqti + caqti-async

A database access layer for OCaml. It provides type-safe functional combinators to interface safely with relational databases. It can mitigate a whole class of injection vulnerabilities at compile time.

### Link please?
https://github.com/paurkedal/ocaml-caqti/
